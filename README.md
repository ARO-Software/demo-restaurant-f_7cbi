# الهنيني · بيت الشاورما — Demo Site

**Business:** مطعم شاورما الهنيني (Shawarma Al-Hunaini) — ضاحية الأمير حسن، عمّان
**Category:** restaurant (shawarma / grill / sandwiches / sweets)
**pitch_type:** site
**Build date:** 2026-07-05
**Demo URL:** https://aro-software.github.io/demo-restaurant-f_7cbi/

Static, single-page, mobile-first Arabic (RTL) demo. Appetite-driven art direction
derived from the restaurant's own signage — charred-ink base, brand crimson-magenta +
ember-orange accents, with the house/gable "roof" from the Hanini logo as the recurring
signature. Sections: hero · about · menu (شاورما / سندويشات / مشاوي / مقبلات ومشروبات /
حلويات) · gallery · hours + Google Maps embed · contact with a WhatsApp CTA to the
business's own number. Photos are the venue's real Google Maps images (local, not
hotlinked). Menu blends verified items/prices (shawarma, sweets) with a plausible
representative grill selection.

Tech: plain HTML/CSS/JS, no backend, no trackers. IntersectionObserver scroll-reveal,
sticky condensing header, respects `prefers-reduced-motion`.
